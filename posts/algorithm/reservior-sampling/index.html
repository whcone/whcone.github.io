水塘抽样算法（Reservior Sampling）

Posted by Hc on Tue, Apr 26, 2022

给一个超大的文件，不可能一下子加载到内存中；文件说上存储着一系列的随机数字， 给定一个数，假定它必定存在与这些随机数字中，需要找到这个数的索引，如果是多 个索引，每次查找这个数时，要等概率地返回任意一个索引。

水塘抽样算法：

逐步的读入文件，遍历随机数组，记录当前索引值，当第 i 次遇到目标数字时， 在区间 [0,i)内随机选择一个整数，如果其等于 0，则将需要返回的索引更新 为当前索引，否则不更新，直到遍历完整个数组。

这样的算法能保证每个索引能等概率的被返回；设数组中目标数字存在 k 个，则：

1 P(第i次遇到值为目标数字的下标成为最终返回的下标)
2=P(第i次随机选择的数字为0) * P(第i+1次随机选择的数字不为0)*...*P(第k次随机选择的数字不为0) 
3=1/i * (1-1/(i+1))*...*(1-1/k)
4=1/i * i/(i+1) * (k-1)/k
5=1/k

 2cnt := 0
 3for i, num := range nums {
 4    if num == target {
 5        cnt++
 6        if rand.Intn(cnt) == 0 {
 7            ans = i
 8        }
 9    }
10}
11return ans
